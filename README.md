# RAQAPathways Website

Static website for RAQAPathways.

## Files
- `index.html` - website content and structure
- `css/style.css` - website styling
- `js/script.js` - mobile menu behavior
- `images/logo.png` - logo

## Deployment
This repository is connected to Cloudflare Pages. Changes committed to the `main` branch will automatically deploy to raqapathways.com.
